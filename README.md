# TaskFlow — Task Management Console Application

A command-line task management application built with C# and .NET 8, 
designed to help users organize their daily tasks efficiently with 
priority levels, due dates, and persistent storage.

## 🚀 Features

- ✅ Add tasks with title, priority, and due date
- ✅ Mark tasks as complete or incomplete
- ✅ Delete tasks with confirmation prompt
- ✅ View all tasks sorted by priority and status
- ✅ Color-coded priority levels (High, Medium, Low)
- ✅ Overdue task detection
- ✅ Data persists between sessions via JSON file storage

## 🛠️ Tech Stack

- **Language:** C# 12
- **Framework:** .NET 8
- **Storage:** JSON file persistence using System.Text.Json
- **IDE:** Visual Studio 2022
- **Architecture:** Repository Pattern
